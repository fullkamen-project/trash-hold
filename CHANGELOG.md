# 📜 История изменений проекта (Changelog)

Все заметные изменения, внесенные в проект `fullkamengg`, будут документироваться в этом файле.
Формат основан на Keep a Changelog.

## [1.3.0] - Расширение хаба и динамическая архитектура

### 🚀 Добавлено (Added)
- Новая карточка для игры "Arena Breakout: Infinite" (ABI) с уникальной темой `.theme-abi` и терракотовым акцентом (`#a34132`).
- Инновационная система динамических SVG-масок для логотипов игр. Теперь одноцветные SVG (FRAGO, ABI) автоматически перекрашиваются в цвета темы при наведении с помощью CSS `mask-image`.
- Глобальный логотип проекта (ЦТА) перемещен на главную страницу (над карточками) с плавной анимацией появления и адаптивным размером (120px для мобильных, 160px для десктопов).

### ♻️ Изменено (Changed)
- Оптимизация производительности: многослойные логотипы EFT переведены из PNG в оптимизированный формат WebP.
- Верстка подписей (subtitles) в карточках: текст теперь выровнен по центру, использует адаптивный размер шрифта (10px на мобильных, 12px на десктопах) и динамически подхватывает акцентный цвет темы при ховере.
- Улучшено центрирование логотипов внутри карточек за счет `flex-shrink-0` и явной передачи размеров `width`/`height` в `next/image`.

## [1.2.0] - 2024-07-26 - Полировка интерактива

### 🚀 Добавлено (Added)
- Внедрена кинематика подъема карточки на `28px` при наведении с плавной анимацией `300ms`.
- Реализована нижняя радиальная подсветка (`h-[28px]`) с использованием акцентного цвета `var(--primary)`.
- Добавлена высокотехнологичная градиентная обводка (`1px`) с использованием CSS-маски (`mask-composite: exclude`) для идеального прилегания к скругленным углам.

## [1.1.0] - Глобальный тактический редизайн

### 🚀 Добавлено (Added)
- Интегрированы кастомные веб-оптимизированные шрифты `BlenderPro-Book` и `BlenderPro-Medium` в форматах `.woff2` и `.woff`.
- Внедрен тактический экран предзагрузки хаба (Preloader) на чистом CSS/Tailwind с анимацией трех квадратов и капс-подписью.
- Внедрена расширяемая архитектура данных `GAMES_DATA` для автоматического рендеринга игровых карточек через метод `.map()`.
- Реализовано состояние `isInactive` для карточек: активирует заглушку "техработ", блокирует ховер-эффекты и роуты, подхватывает специальную текстуру `bg-inactive.webp`.
- Реализована логика бесшовного многослойного видео-ховера (Cross-fade) с аппаратной защитой в виде бэкап-изображения `bg-hover.webp` на случай медленной загрузки видео.

### ♻️ Изменено (Changed)
- Геометрия сайта полностью переведена на строгую модульную сетку из Figma Layout Guide (колонки по `160px`, отступ `28px`).
- Проведен тотальный рефакторинг и оптимизация нейминга кодовой базы: локальный тег "tarkov" заменен на краткий "eft" (в путях папок, ассетах и ID). Адрес внешнего API `https://api.tarkov.dev/graphql` изолирован и сохранен без изменений.
- Базовый CSS-класс `.tarkov-card-base` переименован в универсальный `.tactical-card-base` для поддержки новых игровых разделов.
